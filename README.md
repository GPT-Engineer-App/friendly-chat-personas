# friendly-chat-personas

An AI chat app similar to character.ai that you can choose different personas and start chatting with them. it has a beautiful Airbnb like design. It's built with NextJS, typescript, framer motion, and any other useful frameworks. For it's Chat it uses openAI or groq APIs.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/friendly-chat-personas.git
cd friendly-chat-personas
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
